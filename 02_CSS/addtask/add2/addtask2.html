Блок навигации
  • Ширина 15em
  • Должени присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните достаточным контентом, чтобы по высоте он начинал захватвать вторую статью, как показано в данном примере


Статья 1

Размер блока:
Занимает всю доступню ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным 2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступню ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным 2em, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступню ширину. Высота определяется помещенным внутрь контентом.
